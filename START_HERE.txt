🚀 HOW TO DEPLOY (3 SIMPLE STEPS):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Deploy Backend (10 minutes)
────────────────────────────────────
cd backend
bash deploy.sh

The script will:
✓ Check all files are present
✓ Initialize git repository
✓ Push to Hugging Face Spaces
✓ Show you what secrets to add

Then add secrets in HF Space Settings:
- DATABASE_URL (get from Neon.tech - free)
- OPENAI_API_KEY
- TWILIO credentials (for WhatsApp)
- GMAIL credentials (for Email)
- APP_ENV=production
- API_PORT=7860

Your backend URL:
https://YOUR_USERNAME-taskflow-ai-backend.hf.space


STEP 2: Deploy Frontend (3 minutes)
────────────────────────────────────
1. Go to https://vercel.com
2. Click "Add New Project"
3. Import your GitHub repo
4. Set Root Directory: frontend
5. Add env var: NEXT_PUBLIC_BACKEND_URL = (your HF Space URL)
6. Click "Deploy"

Your frontend URL:
https://your-project.vercel.app


STEP 3: Connect Twilio (1 minute)
──────────────────────────────────
1. Go to https://console.twilio.com/
2. Messaging → WhatsApp Sandbox
3. Set webhook: https://YOUR_USERNAME-taskflow-ai-backend.hf.space/api/webhooks/whatsapp
4. Method: POST
5. Save
